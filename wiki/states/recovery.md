# Recovery — 복구

## 화면 특징
- 특정 화면 없음 (오류/멈춤 등 다양한 상황에서 진입)
- 봇이 현재 상태를 판단할 수 없을 때 진입

## 감지 방법
- 예상 상태와 실제 화면이 불일치
- 알 수 없는 UI 또는 응답 없음
- 이전 액션 실패 반복

## 전이 조건

| 조건 | 대상 상태 |
|------|----------|
| 복구 성공 (상태 재감지) | 이전 상태 (sailing/in-port 등) |
| 3회 시도 실패 | stopped |

## 복구 전략

### 1단계: 화면 재감지
- 현재 화면 스크린샷 → 상태 판별 재시도
- 판별 성공 시 해당 상태로 복귀

### 2단계: 뒤로가기 연타
- 백버튼 3~5회 연속 탭
- 홈 화면 또는 항구 화면으로 돌아가기 시도
- 화면 재감지

### 3단계: 앱 재시작
- 앱 강제 종료 → 재실행
- 로그인 화면 → 로그인 → 캐릭터 선택
- 게임 내 진입 후 상태 재감지

## 주의사항
- 각 단계 실패 시 다음 단계로 진행 (최대 3회 시도)
- 3회 모두 실패 시 stopped 전이 → 사람 개입 요청
- 복구 시도마다 로그 기록 (시도 횟수, 실패 원인)
- 네트워크 오류 시 연결 상태 확인 후 재시도
- 점검/업데이트 중이면 복구 불가 → stopped 즉시 전이
