# Sailing — 항해 중

## 화면 특징
- 바다 배경 (파란색/녹색 톤의 해면)
- 좌상단 미니맵 (현재 위치, 목적지 방향)
- 나침반 및 풍향/풍속 표시
- 좌측 하단 해역 조건 메뉴 (내파, 쇄빙, 돌파 등)
- 함대 속도, 보급품(물/식량/자재/포탄) 잔량 표시
- 자동항해 시 목적지까지 경로선 표시

## 감지 방법
- 바다 배경 + 미니맵 좌상단 존재 → sailing 상태
- 자동항해: 경로선 + 목적지 아이콘 표시, 화면 터치 불필요
- 수동항해: 경로선 없음, 방향 조작 UI 활성

## 전이 조건

| 조건 | 대상 상태 |
|------|----------|
| 항구 근접 (입항 거리 이내) | in-port |
| NPC 함대 조우 (관찰 모드 진입) | event |
| 해적/적 함대 강습 | combat |
| 폭풍우/폭설 등 악천후 진입 | sailing-storm |
| 재해 발생 (화재, 침수, 반란 등) | event |
| 팝업 출현 | event |
| 보급품 고갈 (물/식량 0) | recovery |

## 주의사항
- 자동항해 중 오프라인이 되어도 목적지까지 항해 계속 (부관 위임 설정 필요)
- 풍속/풍향/해류/파도 높이가 항해 속도에 영향
- 날씨 효과: 맑음 -10%, 비 -10%, 폭우 -20%, 폭풍우 -30% 속도 감소
- 흐림/폭우/폭풍우 시 입항 거리 감소 (-300~-500)
- 일부 해역은 선단 LV, 내파, 쇄빙, 돌파, 조력 조건 필요
- 조건 미충족 시 진입 불가 또는 선박 피해 → 항해 불능 위험
