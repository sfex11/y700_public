# Trade — 교역 중

## 화면 특징
- 교역소 화면: 교역품 목록, 가격, 시세(%) 표시
- 매입/매도 탭 구분
- 두카트 잔액 표시
- 적재량 게이지 (선창 용량)
- 교역품별 시세 동향 그래프 (최근 1주일)

## 감지 방법
- "매입"/"매도" 탭 또는 버튼 존재
- 교역품 리스트 + 가격(두카트) + 시세(%) 표시
- 적재량 바 표시

## 전이 조건

| 조건 | 대상 상태 |
|------|----------|
| 매입 완료 (구매 확인) | in-port |
| 매도 완료 (판매 확인) | in-port |
| 뒤로가기 / 교역소 나가기 | in-port |
| 흥정 이벤트 발생 | event |
| 적재량 초과 오류 | recovery |
| 두카트 부족 | recovery |

## 하위 상태
- **trade-buy**: 매입 탭 활성, 구매 수량 선택 중
- **trade-sell**: 매도 탭 활성, 판매 수량 선택 중
- **trade-confirm**: 거래 확인 팝업 (최종 가격, 관세 표시)

## 주의사항
- 시세 100% 기준: 이하=저렴(매입 적합), 이상=비쌈(매도 적합)
- 90% 이하 추천 구매, 110% 이상 추천 판매
- 교역 이벤트: 폭등, 폭락, 유행, 과잉, 폭증 → 시세 급변
- 관세 적용: 자국 외 타국 거래 시 관세 발생, 이중 관세 주의
- 언어 LV이 높을수록 할인/할증 혜택
- 이슬람 문화권: 주류 거래 불가
- 계절에 따라 구매 가능 수량 변동 (계절 마크 확인)
- 협상 전략 전문지식 → 흥정 이벤트 발생 확률 증가
