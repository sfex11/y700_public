# 일반 팝업 닫기

## 목적
- 보상 알림, 팁, 공지 등 일반 팝업을 닫는다
- 확인/닫기 버튼으로 처리한다

## 전제조건
- 상태: 임의 화면 위에 팝업 오버레이 표시
- 화면: 확인 또는 닫기 버튼이 보이는 팝업

## 단계
1. 감지: 팝업 오버레이 + 확인/닫기 버튼 존재 확인
2. 실행: 확인 또는 닫기 버튼 tap
3. 확인: 팝업 사라짐 확인, 이전 화면 복귀

## 실패 시
- 재시도: 버튼 tap 후 팝업 잔존 시 1초 후 재tap (최대 3회)
- 복구: X 버튼 또는 back 버튼으로 닫기 시도

## 타이밍
- 대기: 500ms
- 랜덤 딜레이: 100~400ms

## ADB 명령 예시
- tap(540, 1500): 확인 버튼
- tap(540, 1600): 닫기 버튼
- tap(900, 400): X 버튼 (우측 상단)

## 관련 state
- popup_visible: true
