# 오류 팝업 대응

## 목적
- 접속 오류, 서버 오류 팝업을 감지하고 대응한다

## 전제조건
- 상태: 게임 실행 중 오류 발생
- 화면: 오류 메시지 팝업 표시됨

## 단계
1. 감지: "접속 오류", "서버 오류", "네트워크" 등 오류 텍스트 확인
2. 실행: 확인/재시도 버튼 tap
3. 확인: 팝업 사라짐 → 정상 화면 복귀 여부 판단

## 실패 시
- 재시도: 동일 오류 반복 시 reconnect 액션으로 전환
- 복구: 3회 이상 반복 시 restart-app 액션 실행

## 타이밍
- 대기: 2000ms (서버 응답 대기)
- 랜덤 딜레이: 500~1500ms

## ADB 명령 예시
- tap(540, 1400): 확인/재시도 버튼
- tap(540, 1200): 재접속 버튼

## 관련 state
- error_count: 누적 오류 횟수
- last_error_time: 마지막 오류 발생 시각
- popup_type: error
