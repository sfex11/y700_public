# 시세 확인

## 목적
- 교역소 물품의 현재 시세(%)를 확인하여 매입/매도 판단 근거를 수집한다
- 회계실을 통해 인근 도시 시세도 비교 가능

## 전제조건
- 상태: 교역소 진입 완료 또는 월드맵에서 회계실 사용
- 화면: 교역소 물품 목록 또는 회계실 UI

## 단계
1. 감지: 물품 목록에서 각 아이템 옆 시세(%) 수치 OCR 판독
2. 실행: 목록 스크롤하며 전체 품목 시세 수집. 이벤트 마크 확인 (폭등/폭락/유행/과잉/폭증)
3. 확인: 매입 기준(≤90%) 또는 매도 기준(≥110%) 충족 품목 식별. 계절 마크 여부 체크

## 실패 시
- 재시도: OCR 실패 시 화면 캡처 재시도 (최대 2회)
- 복구: 교역소 재진입 후 목록 새로고침

## 타이밍
- 대기: 800ms (목록 로드)
- 랜덤 딜레이: 150~400ms (스크롤 간격)

## ADB 명령 예시
- swipe(540, 1400, 540, 600): 물품 목록 아래로 스크롤
- tap(900, 300): 회계실 버튼 (월드맵에서)

## 관련 state
- market_prices: {item: price_percent} 딕셔너리
- trade_events: 현재 활성 교역 이벤트 목록
- season_items: 계절 영향 받는 품목
