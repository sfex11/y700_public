# 적재량 확인

## 목적
- 현재 선박의 적재량(cargo capacity)과 잔여 공간을 확인한다
- 물/식량/자재/포탄/교역품 비율을 파악한다

## 전제조건
- 상태: 선박 보유, 도시 내부 또는 항해 중
- 화면: 선박 정보 화면 또는 출항소 적재 비율 화면

## 단계
1. 감지: 선박 정보 아이콘 또는 출항소 적재 비율 UI 확인
2. 실행: 적재량 수치 OCR 판독 (현재/최대). 각 항목(물/식량/자재/포탄/교역품) 비율 수집
3. 확인: 교역품 적재 가능 여유 공간 계산. 교역 특화 선박 여부 확인

## 실패 시
- 재시도: OCR 실패 시 화면 캡처 재시도
- 복구: 출항소에서 비율 조정 후 재확인. 선실 개조 가능 여부 체크

## 타이밍
- 대기: 500ms (UI 로드)
- 랜덤 딜레이: 100~300ms

## ADB 명령 예시
- tap(100, 50): 선박 정보 아이콘
- swipe(540, 1200, 540, 600): 적재 상세 스크롤

## 관련 state
- cargo_max: 최대 적재량
- cargo_current: 현재 적재량
- cargo_ratio: {water, food, material, ammo, trade} 비율
- ship_type: 선박 종류 (교역 특화 여부)
