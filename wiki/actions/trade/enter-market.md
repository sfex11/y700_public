# 교역소 진입

## 목적
- 도시 내 교역소(market)에 진입하여 교역 화면을 연다

## 전제조건
- 상태: 도시 내부 (city_entered = true)
- 화면: 도시 메인 화면 (건물 목록 또는 도시 전경)
- 해당 도시 언어 지식 보유 권장 (언어 LV 높을수록 할인/할증 유리)

## 단계
1. 감지: 도시 메인 화면에서 "교역소" 건물 아이콘 탐색
2. 실행: 교역소 아이콘 tap
3. 확인: 교역소 내부 UI 로드 완료 (매입/매도 탭, 물품 목록 표시)

## 실패 시
- 재시도: 아이콘 미감지 시 화면 스크롤 후 재탐색 (최대 3회)
- 복구: 도시 메인 화면으로 돌아가 재시도. 언어 부족 경고 팝업 시 "확인" tap

## 타이밍
- 대기: 1500ms (건물 진입 로딩)
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(540, 960): 교역소 건물 아이콘 (도시마다 위치 상이)
- tap(540, 1600): 언어 부족 경고 확인 버튼

## 관련 state
- city_entered: 현재 도시 진입 여부
- current_city: 현재 도시명
- language_lv: 해당 도시 언어 레벨
