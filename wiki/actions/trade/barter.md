# 물물 교환 (Barter)

## 목적
- 마을에서 교역품을 물물 교환 전용 교역품으로 교환
- 마을 우호도 상승 (60,000 이후 주요 우호도 수단)

## 전제조건
- 상태: In-Port (마을 내부)
- 화면: 마을 화면, 물물 교환 NPC 또는 메뉴 접근 가능
- 교환 재료 교역품 보유, 마을 우호도 60,000+ (일부)

## 단계
1. 감지: 마을 내 물물 교환 아이콘/NPC 감지
2. 진입: 물물 교환 메뉴 tap → 교환 가능 목록 표시
3. 재료 확인: 필요 교역품 보유 여부 확인 (보유량 vs 필요량)
4. 협상 (선택): 우호도 소모하여 수량/가격 협상 (최대 8회)
5. 교환 실행: [교환] 버튼 tap → 교역품 획득 + 우호도 변동
6. 확인: 교환 완료 메시지, 슬롯 소모 확인

## 주의사항
- 마을별 교환 횟수 제한 (우호도 단계에 따라 1~4회, +1 추가 가능)
- 협상 시 우호도 감소 > 교환 획득 우호도면 오히려 손해
- 교환 전략 LV, 협상 전략 LV이 결과에 영향

## 실패 시
- 재료 부족: 교역소에서 재료 매입 후 재시도
- 교환 횟수 소진: 다음 갱신 대기 (재고와 동일 시간)

## 타이밍
- 대기: 800ms (교환 처리)
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(barter_npc): 물물 교환 NPC 선택
- tap(exchange_item): 교환할 품목 선택
- tap(negotiate_btn): 협상 버튼
- tap(confirm_btn): 교환 확인

## 관련 state
- In-Port → 물물 교환 → In-Port
