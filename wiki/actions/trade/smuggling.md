# 밀수 (Smuggling)

## 목적
- 밀수단 교역상을 통해 밀수품 거래로 수익 확보
- 밀수단의 신용 증서 획득 → 밀수단 상점 이용

## 전제조건
- 상태: In-Port (각 국가 본거지)
- 화면: 도시 내부, 밀수단 교역상 NPC 위치 파악
- 밀수단의 문서 조각 보유 (거래 점수 보상에서 획득)

## 단계
1. 감지: 본거지 도시 내부에서 밀수단 교역상 NPC 탐색
2. 접근: NPC tap → 밀수 거래 화면 진입
3. 매입: 밀수품 선택 → 수량 설정 → 구매 (관세 없음)
4. 이동: 다른 본거지로 항해 (거리 판매 배율 낮음 주의)
5. 매도: 해당 본거지의 밀수단 교역상에서 판매
6. 확인: 순이익 확인, 밀수단의 신용 증서 획득

## 주의사항
- 밀수품 보유 시 출항소/출국 관리소 검문 위험
- 검문 실패: 벌금 + 밀수품 몰수 가능
- 거래 시 해당 도시 동맹 국가 평판 감소 + 업보 증가
- 기존 교역품 효과 미적용, 밀수 전용 효과/전문 지식 적용

## 실패 시
- 검문 발생: 척후법 LV 높으면 도주 확률 증가, 협상 전략 LV로 교섭
- 문서 조각 미보유: 거래 점수 보상 누적 후 재시도

## 타이밍
- 대기: 1000ms (NPC 대화)
- 랜덤 딜레이: 300~800ms

## ADB 명령 예시
- tap(smuggler_npc): 밀수단 교역상 선택
- tap(buy_item): 밀수품 매입
- tap(sell_item): 밀수품 매도

## 관련 state
- In-Port → 밀수 거래 → Sailing → In-Port (다른 본거지) → 밀수 매도
