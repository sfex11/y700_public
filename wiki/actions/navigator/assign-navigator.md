# 항해사 배치 (Assign Navigator)

## 목적
- 함대 선박에 항해사를 배치하여 성능 보너스 획득

## 전제조건
- 상태: 선단 편성 메뉴 또는 선거 진입
- 화면: 함대 편성 → 항해사 배치 슬롯
- 배치할 항해사 보유

## 단계
1. 감지: 함대 선택 → 선박 선택 → 항해사 슬롯 표시 확인
2. 판단: 직업별 적합 포지션 매칭 — 조타장→조타, 회계사→회계, 수리공→정비
3. 판단: 성향 효과 확인 — 교역 성향 항해사 배치 시 교역 보너스
4. 판단: 언어 능력 확인 — 해당 해역 언어 레벨에 따라 교역/탐험 보너스
5. 실행: 슬롯 tap → 항해사 목록에서 대상 tap → "배치" 버튼 tap
6. 확인: 시너지 효과 표시 확인 (특정 항해사 조합 시 추가 효과)

## 실패 시
- 재시도: 슬롯 이미 점유 → 기존 항해사 해제 후 재배치
- 복구: 항해사 부족 → hire-navigator 진행

## 타이밍
- 대기: 800ms
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(300, 800): 항해사 슬롯 선택
- tap(540, 600): 항해사 목록에서 선택
- tap(540, 1300): "배치" 버튼

## 관련 state
- fleet_composition: 함대 편성 상태
- navigator_job: 직업 (포지션 적합도)
- navigator_tendency: 성향 (전투/교역/모험)
- language_levels: 언어 능력 목록
- synergy_pairs: 시너지 조합 목록
