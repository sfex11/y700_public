# 항해사 고용 (Hire Navigator)

## 목적
- 여관/이벤트에서 항해사를 영입하거나 계약서 아이템으로 획득

## 전제조건
- 상태: 항구 내 (docked), 여관 또는 인벤토리 접근 가능
- 화면: 여관 → 고용 탭 또는 인벤토리 → 계약서
- 항해사 슬롯 여유 (선단 LV에 따라 최대 수 증가)

## 단계
1. 감지: 여관 → "고용" 탭 또는 인벤토리 → 계약서 확인
2. 판단: 항해사 정보 확인 — 등급(S/A/B/C), 성향(전투/교역/모험), 직업(조타장/회계사/왕국 탐험가/보급장/수리공/조선공), 레벨, 스킬
3. 실행: 대상 항해사 tap → "고용/사용" 버튼 tap
4. 확인: 항해사 목록에 추가 확인. 슬롯 잔여 확인

## 실패 시
- 재시도: 슬롯 부족 → 불필요한 항해사 해고 후 재시도
- 복구: 계약서 부족 → 이벤트/상회 상점에서 획득

## 타이밍
- 대기: 1000ms (고용 처리)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 400): 고용 탭
- tap(540, 700): 항해사 선택
- tap(540, 1300): "고용" 버튼

## 관련 state
- navigator_slots: 현재/최대 항해사 수
- fleet_level: 선단 LV (슬롯 상한 결정)
- contract_items: 보유 계약서 목록
