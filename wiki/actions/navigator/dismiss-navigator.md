# 항해사 해고 (Dismiss Navigator)

## 목적
- 불필요한 항해사를 목록에서 해고하여 슬롯 확보

## 전제조건
- 상태: 항해사 관리 메뉴 진입
- 화면: 항해사 목록
- 대상 항해사가 함대 배치에서 해제된 상태
- 연대기 고유 항해사가 아닐 것

## 단계
1. 감지: 항해사 목록에서 대상 항해사 확인
2. 판단: 편성 여부 확인 → 편성 중이면 먼저 배치 해제
3. 판단: 연대기 고유 항해사 여부 확인 (고유 항해사는 해고 불가)
4. 실행: 대상 항해사 tap → "해고" 버튼 tap
5. 확인: S급 항해사인 경우 추가 확인 팝업 → "확인" tap
6. 확인: 항해사 목록에서 제거 확인. 일부 명성 보상 수령

## 실패 시
- 재시도: 편성 중 → assign-navigator에서 배치 해제 후 재시도
- 복구: 고유 항해사 해고 시도 → 다른 항해사 선택

## 타이밍
- 대기: 800ms
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 600): 대상 항해사 선택
- tap(540, 1300): "해고" 버튼
- tap(650, 1000): 확인 팝업 "확인"

## 관련 state
- navigator_assigned: 배치 여부 (false여야 해고 가능)
- is_chronicle_navigator: 연대기 고유 여부
- navigator_grade: 등급 (S급 시 추가 확인)
