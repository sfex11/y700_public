# 항해사 성장 (Level Up Navigator)

## 목적
- 항해사에게 경험치를 부여하여 레벨업 및 스탯/스킬 강화

## 전제조건
- 상태: 항해사 관리 메뉴 진입
- 화면: 항해사 상세 → 성장 탭
- 성장 재료 또는 경험치 보유

## 단계
1. 감지: 항해사 상세 화면 → "성장" 탭 선택 확인
2. 실행: 성장 재료 선택 → 수량 조절 → "사용" 버튼 tap
3. 확인: 경험치 바 증가 확인. 레벨업 시 스탯 증가 + 새 스킬 해금 알림
4. 판단: 한계돌파 가능 여부 확인 — 동일 항해사 카드 보유 시 한계 레벨 해제
5. 실행 (한계돌파): "한계돌파" 버튼 tap → 동일 카드 투입 → 확인
6. 확인: 최대 레벨 상한 증가 표시

## 특화 전문 지식 성장
- 활동별 전문 지식 경험치: 박물학, 심미학, 충파술, 포격술 등
- 항해/전투/교역/탐험 활동에 따라 자동 성장

## 실패 시
- 재시도: 재료 부족 → 이벤트/상회 상점에서 성장 재료 확보
- 복구: 한계돌파 카드 부족 → 가챠/이벤트에서 동일 항해사 획득

## 타이밍
- 대기: 1000ms (레벨업 애니메이션)
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 400): 성장 탭
- tap(540, 800): 성장 재료 선택
- tap(700, 1000): 수량 증가
- tap(540, 1300): "사용" 버튼

## 관련 state
- navigator_level: 현재 레벨
- navigator_max_level: 한계 레벨
- growth_materials: 성장 재료 보유량
- duplicate_cards: 동일 항해사 카드 수
