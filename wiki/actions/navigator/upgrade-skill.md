# 스킬 승급 (Upgrade Skill)

## 목적
- 항해사 스킬을 레벨업하여 효과 수치 상승

## 전제조건
- 상태: 항해사 관리 메뉴 진입
- 화면: 항해사 상세 → 스킬 탭
- 스킬북 아이템 보유

## 단계
1. 감지: 항해사 상세 → "스킬" 탭 선택 확인
2. 판단: 스킬 목록 확인 — 등급(일반/고급/희귀/전설), 현재 레벨
3. 판단: 스킬 종류 확인
   - 제독 명령: 함대 충파 강화, 함대 장악, 철벽, 경계 등
   - 전투 스킬: 광역 충파, 정밀 포격, 필사 공격 등
4. 실행: 대상 스킬 tap → 스킬북 투입 → "승급" 버튼 tap
5. 확인: 스킬 레벨 증가 + 효과 수치 상승 표시

## 실패 시
- 재시도: 스킬북 부족 → 이벤트/상회 상점/일일 퀘스트에서 획득
- 복구: 잘못된 스킬 승급 시 되돌리기 불가 — 신중히 선택

## 타이밍
- 대기: 800ms (승급 애니메이션)
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 400): 스킬 탭
- tap(540, 700): 대상 스킬 선택
- tap(540, 1000): 스킬북 선택
- tap(540, 1300): "승급" 버튼

## 관련 state
- skill_list: 항해사 보유 스킬 목록
- skill_level: 각 스킬 현재 레벨
- skill_books: 보유 스킬북 수량
