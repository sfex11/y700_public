# 챕터 진행 (Progress Chapter)

## 목적
- 연대기 메뉴에서 현재 챕터를 확인하고 스토리를 진행한다

## 전제조건
- 상태: 연대기 메뉴 접근 가능
- 화면: [메뉴] → [연대기]
- 이전 챕터 클리어 완료

## 단계
1. 감지: 연대기 메뉴 진입 → 현재 제독(조안 페레로/피리 레이스/레온 페레로 등) 선택
2. 판단: 현재 챕터 확인, 클리어 조건 확인 — 특정 도시 방문/전투 승리/아이템 수집/발견물 보고 등
3. 실행: 챕터 시작 또는 진행 중 퀘스트 확인 tap
4. 판단: 분기점 선택지 등장 시 선택 (이후 스토리 변화에 영향)
5. 실행: 클리어 조건 수행 (해당 액션으로 분기)
6. 확인: 챕터 보상 수령 — 두카트, 경험치, 항해사 계약서, 장비

## 실패 시
- 재시도: 전투 조건 실패 → 함대 강화 후 재도전
- 복구: 조건 미충족 → 해당 도시/발견물 탐색

## 타이밍
- 대기: 1500ms (스토리 로딩)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 300): 연대기 메뉴
- tap(540, 600): 제독 선택
- tap(540, 900): 챕터 진행 버튼
- tap(400, 1100): 분기 선택지 A / tap(680, 1100): 선택지 B

## 관련 state
- current_admiral: 현재 제독
- chapter_id: 현재 챕터 번호
- chapter_conditions: 클리어 조건 목록
- branch_choices: 분기 선택 이력
