# 스토리 완료 (Complete Story)

## 목적
- 챕터 마지막 퀘스트를 완료하고 챕터를 클리어한다

## 전제조건
- 상태: 챕터 내 모든 퀘스트 완료
- 화면: 챕터 클리어 팝업 또는 연대기 메뉴

## 단계
1. 감지: 챕터 클리어 팝업 표시 확인
2. 실행: 클리어 보상 "수령" 버튼 tap
3. 확인: 보상 일괄 수령 — 두카트, 경험치, 아이템 등
4. 판단: 다음 챕터 개방 조건 확인 (일부는 선단 LV 조건)
5. 판단: 엔딩 분기 확인 — 이전 선택지에 따른 다른 결말 표시
6. 확인: 다음 챕터 잠금 해제 표시

## 실패 시
- 재시도: 보상 수령 팝업 미표시 → 연대기 메뉴에서 수동 확인
- 복구: 다음 챕터 선단 LV 부족 → 선단 연구/레벨업 진행

## 타이밍
- 대기: 2000ms (클리어 연출)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 1200): 보상 "수령" 버튼
- tap(540, 1400): "다음 챕터" 버튼
- tap(540, 960): 엔딩 연출 스킵

## 관련 state
- chapter_cleared: 클리어 여부
- next_chapter_requirement: 다음 챕터 조건
- ending_branch: 엔딩 분기 결과
- rewards_collected: 보상 수령 완료 여부
