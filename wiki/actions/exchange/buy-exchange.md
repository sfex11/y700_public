# 거래소 구매 (Buy Exchange)

## 목적
- 거래소에서 레드젬으로 선박/도면/계약서/장비 등을 구매한다

## 전제조건
- 상태: 메뉴 접근 가능
- 화면: [메뉴] → [거래소]
- 레드젬 보유

## 단계
1. 감지: 거래소 메뉴 진입 확인
2. 판단: 카테고리 선택 — 선박, 도면, 계약서, 부품, 장비, 도구
3. 판단: 시세 확인 — 최근 28일 기준 최저가/최고가/평균가 비교
4. 판단: 선박 구매 시 스펙 비교 필터 활용 (최대 4개 비교)
5. 실행: 대상 아이템 tap → 가격 확인 → "구매" 버튼 tap
6. 확인: 레드젬 차감 + 아이템 창고/선거로 이동 확인

## 실패 시
- 재시도: 레드젬 부족 → 판매 등록으로 레드젬 확보
- 복구: 매물 없음 → 시간 경과 후 재검색

## 타이밍
- 대기: 1500ms (서버 통신)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 300): 카테고리 선택
- tap(540, 700): 아이템 선택
- tap(750, 500): 시세 확인 버튼
- tap(540, 1300): "구매" 버튼

## 관련 state
- red_gems: 보유 레드젬
- exchange_listings: 현재 매물 목록
- price_history: 28일 시세 이력
