# 자동 항해 (Auto-Sail)

## 목적
- 목적지까지 자동 항해 실행
- 부관 위임 설정으로 오프라인 항해 가능하게 전환

## 전제조건
- 상태: 항구 화면 또는 해상 화면
- 화면: 월드맵 접근 가능, 목적지 설정 가능
- 물/식량 충분 (목적지까지 소모량 이상)

## 단계
1. 감지: 현재 위치 확인 (좌표 또는 항구명 OCR)
2. 실행: 월드맵 열기 > 목적지 항구 탭 > "자동 항해" 버튼 탭
3. 확인: 항해 중 상태바 출현 확인 (배 아이콘 이동 중)
4. 부관 위임: 부관 위임 버튼 탭 > 위임 확인 팝업 "확인" 탭
5. 확인: 위임 완료 토스트 메시지 감지

## 실패 시
- 재시도: 목적지 미설정 시 월드맵 재진입 후 3회 재시도
- 복구: 보급품 부족 팝업 감지 시 > 항구 보급 액션 호출

## 타이밍
- 대기: 1500ms (월드맵 로딩)
- 랜덤 딜레이: 200~500ms (탭 간격)

## ADB 명령 예시
- tap(540, 1600): 월드맵 버튼
- tap(900, 300): 목적지 항구 선택
- tap(540, 1800): "자동 항해" 버튼
- tap(540, 1750): "부관 위임" 버튼
- tap(640, 1100): 위임 확인 팝업 "확인"

## 관련 state
- state.sailing.mode = "auto"
- state.sailing.destination
- state.sailing.delegated = true/false
- state.supplies.water, state.supplies.food
