# 내구도 관리 (Handle Durability)

## 목적
- 선박 내구도 임계치 이하 시 수리 실행
- 기함 내구도 0 도달 방지 (0 = 항해 불가)

## 전제조건
- 상태: 해상 항해 중 또는 항구 정박 중
- 화면: HUD 내구도 게이지 또는 선박 정보 화면 접근 가능
- 자재(자재) 보유 (해상 수리 시)

## 단계
1. 감지: 내구도 게이지 OCR (HUD 상단)
   - 위험: < 20% (빨간색)
   - 주의: < 40% (노란색)
   - 정상: >= 40%
2. 판단: 수리 방법 결정
   - 항구 정박 중 > 조선소 수리 (자재 불필요, 비용 지불)
   - 해상 + 자재 보유 > 긴급 수리 (자재 소모)
   - 해상 + 자재 없음 > 가장 가까운 항구로 회항 (detect-port 호출)
3. 실행-항구수리: 조선소 탭 > "수리" 버튼 > 확인
4. 실행-해상수리: 선박 메뉴 > "수리" > 자재 사용 확인
5. 확인: 내구도 수치 증가 확인

## 실패 시
- 재시도: 수리 버튼 반응 없음 시 1000ms 후 재탭
- 복구: 골드 부족 팝업 시 > 최소 수리만 실행 또는 스킵

## 타이밍
- 대기: 1000ms (조선소 화면 로딩)
- 랜덤 딜레이: 200~400ms
- 내구도 체크 주기: 10000ms

## ADB 명령 예시
- tap(540, 800): 조선소 메뉴
- tap(540, 1200): "수리" 버튼
- tap(640, 1100): 수리 확인 "확인"
- tap(100, 300): 해상 선박 메뉴 열기
