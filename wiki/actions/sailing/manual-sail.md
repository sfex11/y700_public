# 수동 항해 (Manual Sail)

## 목적
- 바람 방향에 따라 돛 종류 전환하여 최적 속력 유지
- 특수 해역(쇄빙/돌파/조력/내파) 통과 시 수동 조작

## 전제조건
- 상태: 해상 항해 중 (자동 항해 아님)
- 화면: 항해 HUD 표시 중, 바람 방향 아이콘 확인 가능
- 선박 내구도 > 0, 선원 체력 > 0

## 단계
1. 감지: 바람 방향 아이콘 OCR (순풍/측순풍/측역풍/역풍)
2. 실행-돛전환:
   - 순풍/측순풍 > 가로돛 선택 (속도 보너스)
   - 측역풍/역풍 > 세로돛 선택 (속도 보너스)
3. 감지: 특수 해역 진입 알림 (쇄빙/돌파/조력/내파 텍스트)
4. 실행-특수: 필요 스탯 미달 시 우회 경로 스와이프
5. 확인: 속력 수치 변화 확인 (HUD 좌측 상단)

## 실패 시
- 재시도: 돛 전환 실패 시 1000ms 후 재탭
- 복구: 특수 해역 스탯 미달 시 자동 항해로 전환하여 우회

## 타이밍
- 대기: 500ms (돛 전환 애니메이션)
- 랜덤 딜레이: 100~300ms
- 바람 체크 주기: 5000ms

## ADB 명령 예시
- tap(150, 400): 돛 종류 전환 버튼
- tap(300, 500): 가로돛 선택
- tap(300, 600): 세로돛 선택
- swipe(540, 960, 200, 960): 좌측 방향 전환 (우회)

## 관련 state
- state.sailing.mode = "manual"
- state.sailing.wind_direction (순풍/측순풍/측역풍/역풍)
- state.sailing.sail_type (가로돛/세로돛)
- state.sailing.speed
- state.ship.special_stats (쇄빙/돌파/조력/내파)
