# 총리 투표 (Vote Prime)

## 목적
- 주기적 총리 선거 이벤트에 참여하여 투표 + 보상 획득

## 전제조건
- 상태: 총리 선거 이벤트 기간
- 화면: 이벤트 배너 또는 [메뉴] → [총리 선거]
- 투표권 보유 (1일 N회, 아이템으로 추가 가능)

## 단계
1. 감지: 총리 선거 이벤트 배너 또는 메뉴 진입 확인
2. 판단: 후보별 공약 확인 — 세금, 교역 보너스, 전투 보너스 등
3. 실행: 후보 선택 → "투표" 버튼 tap
4. 확인: 투표 완료 → 투표 보상 수령 (두카트, 경험치)
5. 확인: 투표권 잔여 횟수 확인
6. 결과: 당선 시 해당 국가 보너스 적용

## 실패 시
- 재시도: 투표권 소진 → 아이템으로 추가 투표권 사용
- 복구: 이벤트 기간 종료 → 다음 선거 대기

## 타이밍
- 대기: 1000ms
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 400): 선거 메뉴 진입
- tap(540, 700): 후보 선택
- tap(540, 1300): "투표" 버튼

## 관련 state
- election_active: 선거 이벤트 활성 여부
- vote_remaining: 잔여 투표 횟수
- candidate_pledges: 후보별 공약 정보
