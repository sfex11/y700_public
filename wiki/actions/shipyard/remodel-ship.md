# 선박 개조 (Remodel Ship)

## 목적
- 기존 선박의 스펙을 업그레이드한다 (선체/돛/포/특수장비 슬롯)
- 개조 LV 필요, 도면 + 자재 소모

## 전제조건
- 상태: 항구 내 (docked), 조선소 메뉴 진입
- 화면: 조선소 → 개조 탭
- 개조 LV 충족, 도면 + 자재 보유

## 단계
1. 감지: 조선소 → "개조" 탭 선택 확인
2. 실행: 보유 선박 목록에서 대상 선박 tap
3. 판단: 개조 슬롯(선체/돛/포/특수장비) 중 업그레이드 대상 선택
4. 확인: 현재 개조 단계(+1~+10) 확인. 성공 확률 표시 확인
5. 실행: 도면 + 자재 투입 → "개조" 버튼 tap
6. 확인: 성공 시 스펙 상승 표시. 실패 시 재료만 소모, 선박 유지

## 실패 시
- 재시도: 개조 실패 → 재료 재확보 후 재시도 (높은 단계일수록 실패율 증가)
- 복구: 재료 부족 시 거래소/교역으로 확보

## 타이밍
- 대기: 1500ms (개조 애니메이션)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 400): 개조 탭
- tap(540, 600): 대상 선박 선택
- tap(400, 900): 개조 슬롯 선택
- tap(540, 1300): "개조" 버튼

## 관련 state
- remodel_level: 개조 LV
- ship_remodel_stage: 현재 개조 단계 (+0~+10)
- success_rate: 성공 확률 (단계별 감소)
