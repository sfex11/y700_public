# 장비 강화/장착 (Upgrade Equipment)

## 목적
- 선박에 장비(대포/돛/장갑/특수장비) 장착 및 강화

## 전제조건
- 상태: 항구 내 (docked), 조선소 또는 선거 메뉴 진입
- 화면: 장비 관리 탭
- 장비 보유 (제작/거래소 구매/드롭)

## 단계
1. 감지: 선박 선택 → 장비 슬롯 표시 확인
2. 판단: 빈 슬롯 또는 교체 대상 슬롯 선택. 선박 등급에 따라 슬롯 수 상이
3. 실행 (장착): 장비 목록에서 대상 장비 tap → "장착" 버튼 tap
4. 실행 (강화): 장착된 장비 tap → "강화" 버튼 → 재료 + 골드 투입 → 확인
5. 실행 (제작): "제작" 탭 → 도면 선택 → 재료 투입 → "제작" 버튼 tap
6. 확인: 장비 스펙 변화 확인

## 실패 시
- 재시도: 재료 부족 → 거래소/교역으로 확보 후 재시도
- 복구: 잘못 장착 시 탈착 후 재장착

## 타이밍
- 대기: 1000ms (강화 애니메이션)
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(540, 600): 선박 선택
- tap(300, 800): 장비 슬롯 선택
- tap(540, 1000): 장비 목록에서 장비 선택
- tap(540, 1300): "장착/강화/제작" 버튼

## 관련 state
- ship_grade: 선박 등급 (슬롯 수 결정)
- equipment_inventory: 보유 장비 목록
- upgrade_materials: 강화 재료 보유량
