# 전리품 수집 (Collect Reward)

## 목적
- 전투 승리 후 전리품(아이템, 금화, 나포 함선)을 확인하고 수집한다

## 전제조건
- 상태: 전투 종료(combat_end), 승리 판정 완료
- 화면: 전투 결과 화면 표시 (승리 배너 + 전리품 목록)

## 단계
1. 감지: "승리" 배너 및 전리품 결과 화면 표시 확인
2. 확인: 전리품 목록 스크롤하여 획득 아이템/금화 확인
3. 나포: 나포 가능 함선이 있으면 "나포" 버튼 tap → 함대에 편입 또는 매각 선택
4. 수집: "확인" 버튼 tap하여 전리품 최종 수령
5. 복귀: 항해 화면으로 복귀 확인

## 실패 시
- 재시도: 화면 전환 지연 → 3000ms 추가 대기 후 "확인" 버튼 재탐색
- 복구: 함대 슬롯 가득 참 → 나포 포기 또는 기존 함선 해체 후 재시도

## 타이밍
- 대기: 결과 화면 로딩 2000ms
- 랜덤 딜레이: 300~600ms (수집 버튼 tap 전)

## ADB 명령 예시
- tap(540, 1400): "확인" 버튼 (전리품 수령)
- tap(700, 1000): "나포" 버튼 (나포 가능 시)
- tap(540, 1600): 나포 확인 팝업 "편입"
- swipe(540, 1200, 540, 800): 전리품 목록 스크롤

## 관련 state
- combat_end → reward_screen → collect → sailing
- 나포 분기: reward_screen → capture_select → fleet_manage → sailing
