# 전투 개시 (Engage Battle)

## 목적
- 해상에서 적 함대를 발견한 뒤 관찰 모드를 거쳐 전투를 시작한다

## 전제조건
- 상태: 항해 중(sailing), 적 함대가 화면에 표시됨
- 화면: 해상 맵에서 적 함대 아이콘 감지 가능

## 단계
1. 감지: 적 함대 아이콘(붉은 깃발 마커)을 화면에서 탐색
2. 실행: 적 함대 아이콘 tap하여 선택 → 관찰 모드 진입
3. 확인: 관찰 모드에서 적 정보(함대 구성, 거리, 전투 예측) 표시 확인
4. 판단: 전투 예측 결과가 유리하면 "전투" 버튼 tap
5. 확인: 턴제 hex 맵 전투 화면 로딩 완료 확인

## 실패 시
- 재시도: 적이 이동하여 선택 불가 → 접근 후 재탐색 (최대 3회)
- 복구: 전투 예측 불리 시 "도주" 선택 → flee-battle 액션으로 전환

## 타이밍
- 대기: 관찰 모드 로딩 1500ms
- 랜덤 딜레이: 200~500ms (전투 버튼 tap 전)
- 전투 맵 로딩: 3000ms 대기

## ADB 명령 예시
- tap(540, 960): 적 함대 아이콘 선택
- tap(750, 1700): "전투" 버튼 (관찰 모드 하단 우측)
- tap(300, 1700): "도주" 버튼 (관찰 모드 하단 좌측)

## 관련 state
- sailing → encounter → observation → combat_ready
- 날씨 확인 필요: 맑음(포격 +15%, 백병 -15%), 비(이동력 -1, 화재 -20%)
