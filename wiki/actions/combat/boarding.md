# 접현전 / 백병전 (Boarding)

## 목적
- 적 함선에 인접하여 백병 공격으로 선원 체력을 소진시키거나, 나포를 노린다

## 전제조건
- 상태: 전투 중(in_combat), 해당 함선의 턴
- 화면: 아군 함선이 적 함선과 인접(hex 1칸) 상태
- 조건: 백병선이 유리 (백병선 > 포격선 상성)

## 단계
1. 감지: 아군 백병선과 적 함선의 hex 거리가 1인지 확인
2. 이동: 인접하지 않으면 적 함선 옆 hex로 이동 tap
3. 실행: "백병" 공격 버튼 tap → 인접 적 함선 tap
4. 확인: 백병 전투 애니메이션 → 적 선원 체력 감소 확인
5. 결투: 적 선원 체력이 낮으면 "결투" 선택 가능 → 1v1 함장 전투로 즉시 승부

## 나포 조건
- 백병/결투로 승리 시 큰 차이로 이기면 나포 확률 발생
- 적 내구도를 최대한 보존한 채 선원 체력만 깎는 것이 핵심

## 실패 시
- 재시도: 이동력 부족으로 인접 불가 → 다음 턴 접근 (충파선 주의)
- 복구: 백병 피해 과다 시 구급함 사용 또는 포격으로 전환

## 타이밍
- 대기: 백병 애니메이션 2000ms, 결투 애니메이션 3000ms
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(hex_x, hex_y): 이동할 hex 타일 선택
- tap(400, 1800): "백병" 공격 버튼
- tap(enemy_x, enemy_y): 대상 적 함선 선택
- tap(540, 1600): "결투" 버튼 (조건 충족 시 표시)

## 관련 state
- in_combat → move → adjacent → boarding_attack → damage_check
- 도구 활용: 구급함(선원 회복), 소화모래(화재 진압)
