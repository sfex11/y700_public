# 스킬 사용 (Use Skill)

## 목적
- 전투 중 해전 기술 또는 빠른 기술을 사용하여 전투 우위를 확보한다

## 전제조건
- 상태: 전투 중(in_combat), 해당 함선의 턴
- 화면: 스킬 슬롯 UI가 하단에 표시됨
- 조건: 스킬 쿨타임 완료, 필요 자원(사기 등) 충족

## 단계
1. 감지: 현재 턴 함선의 스킬 슬롯(최대 4개 빠른 기술) 활성 상태 확인
2. 판단: 적 위치/상성에 따라 최적 스킬 선택 (포격선→포격 스킬, 백병선→백병 스킬 등)
3. 실행: 스킬 아이콘 tap → 대상 선택(적 함선 또는 hex 타일) tap
4. 확인: 스킬 발동 애니메이션 완료, 데미지/효과 적용 확인
5. 제독명령: 필요 시 사기 소모하여 제독 명령 발동 (턴 소모 없음, 기함만 가능)

## 실패 시
- 재시도: 대상 선택 오류 → 빈 타일 tap으로 취소 후 재선택
- 복구: 쿨타임 미완료 시 일반 공격(포격/백병/충파)으로 전환

## 타이밍
- 대기: 스킬 발동 애니메이션 1500~2500ms
- 랜덤 딜레이: 150~400ms (스킬 선택 전)

## ADB 명령 예시
- tap(200, 1750): 스킬 슬롯 1번
- tap(350, 1750): 스킬 슬롯 2번
- tap(500, 1750): 스킬 슬롯 3번
- tap(650, 1750): 스킬 슬롯 4번
- tap(target_x, target_y): 대상 적 함선 선택

## 관련 state
- in_combat → skill_select → target_select → skill_execute → in_combat
- 협공 조건: 아군이 같은 공격 타입 사거리 내 + 미행동 → 보너스 데미지
