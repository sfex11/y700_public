# 전투 위임 (Delegate Combat)

## 목적
- 전투를 AI에 위임하여 자동 진행
- 약한 적 상대 시 수동 조작 불필요할 때 사용

## 전제조건
- 상태: Event (전투 중)
- 화면: 전투 맵 화면, 위임 버튼 표시

## 단계
1. 감지: 전투 맵 진입 확인 (턴제 전투 UI)
2. 판단: 전투 승률 예측 (관찰 모드에서 확인한 정보 기반)
3. 실행: 화면 좌측/우측 하단 [위임] 버튼 tap
4. 속도 조절: [배속] 버튼 tap (2배속/최대 속도)
5. 대기: 전투 종료까지 대기
6. 확인: 전투 결과 화면 표시 (승리/패배)

## 실패 시
- 전투 패배: 회항 처리 → Recovery 상태
- 위임 해제 필요: 위임 버튼 재tap으로 수동 전환
- 기함 위험(내구도/선원 체력 저하): 수동 전환 → 도구 사용

## 주의사항
- 부관 위임(오프라인): 기함에 부관실 필요, 선단 관리에서 대응 방법 설정
- PvP 위임: 추가 시간 100초 소진 시 자동 위임 전환 (해제 불가)
- 강적 상대 시 위임 비권장 → use-skill.md 참조

## 타이밍
- 대기: 전투 종료까지 (10~120s)
- 체크 간격: 5000ms (전투 종료 여부)

## ADB 명령 예시
- tap(delegate_btn): 위임 버튼
- tap(speed_btn): 배속 버튼
- 전투 종료 감지: 결과 화면 템플릿 매칭

## 관련 state
- Event(전투) → 위임 → 전투 종료 → collect-reward → Sailing/In-Port
