# 항구 입항 (Dock)

## 목적
- 항해 중 항구에 정박하여 항구 UI 진입

## 전제조건
- 상태: 항해 중 (sailing)
- 화면: 월드맵에서 항구 아이콘이 보이는 범위
- 항구 접안 범위 내 (날씨에 따라 범위 변동)

## 단계
1. 감지: 월드맵에서 항구 아이콘 탐지 (이미지 매칭)
2. 실행: 항구 아이콘 tap → 입항 확인 버튼 tap
3. 확인: 로딩 완료 후 항구 UI 요소 (출항소/여관 등) 감지

## 날씨별 접안 범위 보정
- 맑음: 기본 범위
- 흐림: -300 감소
- 폭우: -300 감소
- 폭풍우: -500 감소
- 척후법 지식 보유 시 범위 증가

## 실패 시
- 재시도: 범위 밖이면 항구 방향으로 이동 후 3초 대기, 재시도
- 복구: 3회 실패 시 날씨 대기 또는 다른 항구 탐색

## 타이밍
- 대기: 2000ms (로딩 완료 대기)
- 랜덤 딜레이: 300~800ms (tap 간격)

## ADB 명령 예시
- tap(port_x, port_y): 항구 아이콘 위치
- tap(confirm_x, confirm_y): 입항 확인 버튼

## 관련 state
- sailing → docked
- weather_condition
- port_id
