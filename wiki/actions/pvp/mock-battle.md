# 모의전 참여 (Mock Battle)

## 목적
- PvP 모의전에서 상대 함대와 전투하여 계급 상승 + 보상 획득

## 전제조건
- 상태: 선단 LV 25 이상
- 화면: [메뉴] → [모의전]
- 티켓 보유 (최초 6장, 시간 경과 회복, 30 블루젬 추가 구매)

## 단계
1. 감지: 모의전 메뉴 진입 → 티켓 잔여 확인
2. 실행: 내 함대 + 진형 등록 (미등록 시 편성 먼저)
3. 판단: 상대 5명 표시 → 전투력/계급 비교 → 승리 가능 상대 선택
4. 판단: 전장 종류 확인 — 해안가(순수), 만(백병+30%), 유빙(충파+30%), 섬(포격+30%)
5. 실행: 상대 선택 → "전투" 버튼 tap
6. 확인: 전투 결과 — 승리 시 계급 포인트 획득, 패배 시 감소
7. 참고: 5명 격파 시 새 상대 등장. 상대 갱신 1일 1회 무료

## 계급 체계
- 청동 3급 → 청동 2급 → ... → 전설 (상위 5위)
- 보상: 계급 달성 보상 + 주간 보상

## 실패 시
- 재시도: 패배 → 함대 편성/장비 조정 후 다른 상대 선택
- 복구: 티켓 소진 → 시간 대기 또는 30 블루젬 추가 구매

## 타이밍
- 대기: 3000ms (전투 로딩)
- 랜덤 딜레이: 300~600ms

## ADB 명령 예시
- tap(540, 400): 모의전 메뉴
- tap(540, 600): 함대 등록/편성
- tap(540, 800): 상대 선택
- tap(540, 1300): "전투" 버튼
- tap(540, 960): 전투 결과 확인

## 관련 state
- fleet_level: 선단 LV (25 이상 필수)
- mock_tickets: 잔여 티켓
- pvp_rank: 현재 계급
- battlefield_type: 전장 종류 (보너스 확인)
- opponent_list: 상대 5명 목록
