# 퀘스트 완료

## 목적
- 의뢰(퀘스트) 완료 시 보상을 수령한다
- 퀘스트 유형: 칙명, 여관 의뢰, 조합 의뢰, 돌발 의뢰

## 전제조건
- 상태: 의뢰 목표 달성 완료
- 화면: 퀘스트 완료 알림 또는 의뢰 NPC 대화 화면

## 단계
1. 감지: "의뢰 완료" 배너 또는 NPC 머리 위 완료 아이콘 출현
2. 실행: 완료 버튼 tap → 보상 수령 화면 → 확인 tap
3. 확인: 블루젬/두카트/재료 획득 팝업 표시 확인

## 실패 시
- 재시도: 보상 수령 버튼 미반응 시 1.5초 후 재tap (최대 3회)
- 복구: NPC 대화 진행 불가 시 back → 해당 NPC 재방문

## 타이밍
- 대기: 1000ms (보상 연출 대기)
- 랜덤 딜레이: 200~600ms

## ADB 명령 예시
- tap(540, 1700): 완료/확인 버튼
- tap(540, 960): NPC 대화 진행

## 관련 state
- quest_type: 칙명 | 여관 | 조합 | 돌발
- quest_target_met: true
- 보상 출처: 조합(guild), 여관(inn), 왕궁(palace)
