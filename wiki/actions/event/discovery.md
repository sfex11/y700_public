# 발견물 이벤트

## 목적
- 탐험 중 발견물 획득 이벤트를 처리한다
- 수집가에게 보고하여 명성과 재화를 획득한다

## 전제조건
- 상태: 탐험(exploration) 진행 중
- 화면: 발견물 발견 팝업 표시됨

## 단계
1. 감지: 화면에 발견물 아이콘 및 "발견" 텍스트 출현
2. 실행: 확인 버튼 tap → 발견물 상세 화면 → 수집가 보고 버튼 tap
3. 확인: 명성/재화 획득 알림 표시 확인

## 실패 시
- 재시도: 확인 버튼 미반응 시 2초 후 재tap (최대 3회)
- 복구: 팝업 사라지지 않으면 back 버튼으로 닫기

## 타이밍
- 대기: 1500ms (발견물 연출 완료 대기)
- 랜덤 딜레이: 300~800ms

## ADB 명령 예시
- tap(540, 1600): 확인 버튼
- tap(800, 1400): 수집가 보고 버튼

## 관련 state
- 박물학 지식 수치 → 발견 확률 증가
- 심미학 지식 수치 → 보고 보상 증가
- exploration_active: true
