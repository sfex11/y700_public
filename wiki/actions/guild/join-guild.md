# 상회 가입/생성 (Join Guild)

## 목적
- 상회에 가입하거나 새 상회를 창설한다

## 전제조건
- 상태: 일정 레벨 이상 (초반 불가)
- 화면: [메뉴] → [상회]
- 가입: 기존 상회 미소속. 생성: 창립 비용 (두카트) 보유

## 단계
1. 감지: 상회 메뉴 진입 → 미가입 상태 확인
2. 판단: 가입 vs 생성 결정
3. 실행 (가입): "상회 검색" → 조건 필터 → 대상 상회 tap → "가입 신청" tap
4. 실행 (생성): "상회 생성" → 이름/설명 입력 → 창립 비용 지불 → "생성" tap
5. 확인: 상회 소속 표시 확인. 상회 최대 인원은 상회 LV에 따라 증가

## 탈퇴
- 상회 설정 → "탈퇴" → 24시간 쿨타임 적용

## 실패 시
- 재시도: 가입 거절 → 다른 상회 검색
- 복구: 두카트 부족 (생성) → 교역으로 확보

## 타이밍
- 대기: 1500ms (서버 통신)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 400): 상회 메뉴
- tap(540, 700): 상회 검색/생성 버튼
- tap(540, 1300): "가입 신청/생성" 버튼

## 관련 state
- guild_membership: 소속 상회 (null이면 미가입)
- player_level: 플레이어 레벨
- ducats: 보유 두카트 (생성 시)
- guild_cooldown: 탈퇴 쿨타임 잔여
