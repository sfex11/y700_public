# 상회 미션 (Guild Mission)

## 목적
- 일일/주간 상회 미션을 수행하여 기여도 + 상회 포인트 획득

## 전제조건
- 상태: 상회 소속
- 화면: [메뉴] → [상회] → 미션 탭

## 단계
1. 감지: 상회 → "미션" 탭 선택 → 일일/주간 미션 목록 확인
2. 판단: 미션 타입 확인 — 교역량 달성, 전투 승리, 발견물 보고 등
3. 실행: 수행 가능한 미션 선택 → 해당 활동 수행 (교역/전투/탐험)
4. 확인: 미션 목표 달성 시 "완료" 버튼 활성화 → tap
5. 확인: 개인 기여도 + 상회 포인트 획득 표시
6. 확인: 상회 랭킹 — 주간 기여도 순위 확인

## 실패 시
- 재시도: 미션 기한 내 미달성 → 다음 일일/주간 갱신 대기
- 복구: 미션 조건 불가 → 다른 미션 수행

## 타이밍
- 대기: 1000ms
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 400): 미션 탭
- tap(540, 700): 미션 선택
- tap(540, 1300): "완료" 버튼

## 관련 state
- guild_missions_daily: 일일 미션 목록/진행도
- guild_missions_weekly: 주간 미션 목록/진행도
- contribution_points: 개인 기여도
- guild_points: 상회 포인트
- guild_level: 상회 LV
