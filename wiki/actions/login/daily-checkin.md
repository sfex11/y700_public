# 출석부/일일보상

## 목적
- 출석부(일일보상)를 수령한다
- 일일 과제는 매일 00:00에 초기화된다

## 전제조건
- 상태: 게임 홈 화면 진입 완료
- 화면: 출석부 팝업 자동 표시 또는 홈 화면에서 접근 가능

## 단계
1. 감지: 출석부 팝업 자동 출현 또는 홈 화면 출석부 아이콘 확인
2. 실행: 보상 수령 버튼 tap → 확인 tap → 팝업 닫기
3. 확인: "수령 완료" 상태 표시 또는 보상 획득 알림 확인

## 실패 시
- 재시도: 수령 버튼 미반응 시 1초 후 재tap (최대 3회)
- 복구: 이미 수령한 경우 닫기 처리

## 타이밍
- 대기: 1000ms (보상 연출 대기)
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(540, 1200): 보상 수령 버튼
- tap(540, 1600): 확인 버튼
- tap(900, 300): X 닫기

## 관련 state
- daily_checkin_done: false → true
- daily_reset_time: 00:00
- screen: home → checkin_popup → home
