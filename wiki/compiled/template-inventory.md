## 중요 발견: 보안정책 위반 팝업

세계지도 → **"탐험" 탭** 클릭 시 게임 강제종료.
- 메시지: "보안 정책 위반이 탐지되어 현재 앱을 강제 종료합니다."
- Code: 00000028
- 원인 추정: ADB USB 디버깅 + 자동 입력 감지 (Play Integrity API)
- **항구/항로/교역 탭은 안전** (탐험 탭만 트리거)
- recovery/popup 카테고리에 자동 복구용 템플릿 등록 완료
