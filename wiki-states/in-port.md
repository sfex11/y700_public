# In-Port — 항구 도착

## 화면 특징
- 도시 전경 배경 (건물, 부두, 거리)
- 도시 시설 목록 (지도에서 확인 가능)
- NPC 캐릭터 표시
- 상단 도시 이름 표시

## 감지 방법
- 건물/부두 배경 + 도시 시설 UI 존재
- "교역소", "조선소", "여관", "출항소", "은행" 등 시설명 텍스트
- 월드맵이 아닌 도시 내부 화면

## 전이 조건

| 조건 | 대상 상태 |
|------|----------|
| 교역소 선택 | trade |
| 출항소 → 출항 버튼 | sailing |
| 팝업/이벤트 발생 | event |
| NPC 전투 발생 | combat |
| 오류/멈춤 | recovery |

## 하위 상태
- **port-menu**: 시설 선택 화면 (지도 또는 시설 목록)
- **port-dialog**: NPC 대화 진행 중 (여관 종업원, 조합 의뢰 등)
- **port-supply**: 출항소에서 보급 중 (물/식량/자재/포탄 적재)

## 주의사항
- 도시 시설은 규모에 따라 다름: 여관·교역소·조선소는 모든 도시에 존재
- 도구점, 종교 시설, 왕궁, 수집가 저택은 특정 도시에만 존재
- 출항소에서 선박 수리·선원 모집·물자 보급 가능 (자동 보급 기능)
- 선박 배치/교체는 도시 정박 중에만 가능
- 조선소: 선박 건조, 수리, 해체, 개조, 선실 변경
- 여관: 선원 모집, 항해사 고용, 충성도 관리
