# Combat — 전투

## 화면 특징
- 별도 전투 맵 (육각형 타일 기반)
- 아군/적 함대 배치, 선박별 HP(내구도) 바
- 상단 턴 순서 표시 (기동력 기반)
- 우측 상단 선박 상성표 (돌격↔백병↔포격)
- 하단 전투 행동 버튼: 포격, 백병, 충파, 결투, 수리, 의술
- 우측 하단 빠른 기술 아이콘 (최대 4개)
- 제독 초상화 아래 사기 게이지

## 감지 방법
- 육각형 타일 맵 + 선박 아이콘 배치
- "전투"/"도주"/"항복"/"교섭" 버튼 (전투 성립 단계)
- 턴 순서 바 + HP 바 표시

## 전이 조건

| 조건 | 대상 상태 |
|------|----------|
| 승리 (적 전멸) | sailing |
| 패배 (아군 전멸) | recovery |
| 도주 성공 | sailing |
| 도주 실패 | combat (전투 계속) |
| 항복 선택 | sailing |
| 전투 완료 팝업 | event |

## 전투 유형
- **NPC 전투**: 항해 중 NPC 함대와 교전, 관찰 모드 → 전투 선포
- **해적 전투**: 해적 함대 강습 또는 토벌
- **PvP 전투**: 다른 유저와 실시간 전투 (결투/도구 사용 불가, 도주 불가)

## 주의사항
- 선박 상성: 돌격선 > 백병선 > 포격선 > 돌격선 (지원선은 상성 무관)
- 포격은 원거리, 백병은 인접, 충파는 일직선 방향만 가능
- 협공: 같은 공격 형태의 아군이 범위 내 + 미행동 시 동시 공격
- 날씨 효과: 맑음(포격+15%/백병-15%), 비(기동력-20%/화재-20%) 등
- 물/식량 부족 시 매 턴 사기 감소, 심하면 아군 전선 이탈
- PvP: 턴당 기본 5초 + 추가 100초, 초과 시 자동 위임
- 선원 체력 0 또는 내구도 0 → 선박 침몰
- 패배 시 즉시 부활 불가, 회항만 가능 (PvP)
