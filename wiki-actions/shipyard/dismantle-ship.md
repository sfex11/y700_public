# 선박 해체 (Dismantle Ship)

## 목적
- 불필요한 선박을 해체하여 자재 일부를 회수한다

## 전제조건
- 상태: 항구 내 (docked), 조선소 메뉴 진입
- 화면: 조선소 → 해체 탭
- 대상 선박이 함대 편성에서 제외된 상태
- 장착 장비 모두 탈착 완료

## 단계
1. 감지: 조선소 → "해체" 탭 선택 확인
2. 실행: 해체 가능 선박 목록에서 대상 tap
3. 판단: 장비 장착 여부 확인. 장착 시 먼저 탈착 안내
4. 실행: "해체" 버튼 tap → 확인 팝업 "확인" tap
5. 확인: 자재 회수 결과 표시 확인 (건조 비용의 30~50% 회수)

## 실패 시
- 재시도: 편성 중 선박 → 선거에서 먼저 제외 후 재시도
- 복구: 장비 미탈착 → 장비 관리에서 탈착 후 재시도

## 타이밍
- 대기: 1000ms (해체 처리)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 400): 해체 탭
- tap(540, 600): 대상 선박 선택
- tap(540, 1300): "해체" 버튼
- tap(650, 1000): 확인 팝업 "확인"

## 관련 state
- ship_in_fleet: 편성 여부 (false여야 해체 가능)
- equipped_items: 장착 장비 목록 (비어야 해체 가능)
- recovered_materials: 회수 자재 목록
