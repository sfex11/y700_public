# 선박 건조 (Build Ship)

## 목적
- 조선소에서 새 선박을 건조한다
- 건조 LV와 자재(목재/철물/돛포) + 두카트 소모

## 전제조건
- 상태: 항구 내 (docked), 조선소 메뉴 진입
- 화면: 조선소 → 건조 탭
- 건조 LV 충족 (T1: LV1, T2: LV5, T3: LV10, T4: LV20, T5: LV30, T6: LV40)
- 자재 + 두카트 충분

## 단계
1. 감지: 조선소 → "건조" 탭 선택 확인
2. 실행: 건조 가능 선박 목록에서 원하는 선종(범선/갤리/카락/갤리언/프리깃 등) tap
3. 판단: 자재 보유량 vs 필요량 비교. 부족 시 스킵
4. 실행: "건조 시작" 버튼 tap → 건조 대기 시간 표시
5. 확인: 건조 큐에 등록 확인. 블루젬 즉시완료 여부 판단 (고가 선박만)
6. 완료: 건조 완료 후 선거(선박 관리)에서 편성

## 실패 시
- 재시도: 자재 부족 → 거래소/교역으로 자재 확보 후 재시도
- 복구: 건조 LV 부족 → 하위 티어 선박 건조

## 타이밍
- 대기: 1500ms (메뉴 전환)
- 랜덤 딜레이: 300~600ms
- 건조 대기: T1 즉시~T6 수시간 (블루젬 단축 가능)

## ADB 명령 예시
- tap(540, 400): 건조 탭 선택
- tap(540, 700): 선박 종류 선택
- tap(540, 1300): "건조 시작" 버튼
- tap(700, 1300): 블루젬 즉시완료 버튼 (선택)

## 관련 state
- build_level: 건조 LV
- materials: {wood, iron, sail_cloth}
- ducats: 보유 두카트
- fleet_level: 선단 LV (편성 가능 선박 수 연관)
