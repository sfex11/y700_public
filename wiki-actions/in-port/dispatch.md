# 파견 (Dispatch Fleet)

## 목적
- 2~4함대를 모험/교역/전투 파견 임무에 보내 자동 보상 획득
- 파견 완료 시 보상 수령 및 재파견 처리

## 전제조건
- 상태: In-Port 또는 Sailing (메뉴 접근 가능)
- 화면: [메뉴] → [파견] 접근 가능
- 파견 함대(2~4함대) 편성 완료

## 단계
1. 감지: [메뉴] → [파견] tap → 파견 현황 화면 확인
2. 완료 확인: 파견 완료된 함대에 "완료" 표시 감지
3. 보상 수령: 완료된 함대 tap → 보상 확인 → 수령 tap
4. 재파견: [재파견] 버튼 tap → 동일 임무 재시작
5. 새 파견: 미파견 함대 선택 → 지역 선택 → 임무 선택 → 파견 시작

## 실패 시
- 파견 조건 미달: 선단 LV, 함대 전투력 확인 → 다른 임무 선택
- 함대 미편성: 선박/항해사 배치 필요 → 스킵 후 다음 기회

## 타이밍
- 대기: 1000ms (화면 전환)
- 랜덤 딜레이: 300~700ms

## ADB 명령 예시
- tap(menu_btn): 메뉴 열기
- tap(dispatch_tab): 파견 탭 선택
- tap(fleet_slot): 함대 슬롯 선택
- tap(mission_btn): 임무 선택/시작
- tap(collect_btn): 보상 수령

## 관련 state
- In-Port / Sailing → 파견 메뉴 → 보상 수령/재파견 → 이전 상태
- 파견 중 선박 이해도 증가 안 됨
