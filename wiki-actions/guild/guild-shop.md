# 상회 상점 (Guild Shop)

## 목적
- 상회 포인트로 아이템을 교환 구매한다

## 전제조건
- 상태: 상회 소속
- 화면: [메뉴] → [상회] → 상점 탭
- 상회 포인트 보유

## 단계
1. 감지: 상회 → "상점" 탭 선택 확인
2. 판단: 품목 확인 — 자재, 장비, 계약서, 도면 등. 상회 LV에 따라 품목 확장
3. 판단: 일일/주간 구매 한도 잔여 확인
4. 실행: 대상 아이템 tap → 수량 선택 → "교환" 버튼 tap
5. 확인: 상회 포인트 차감 + 아이템 획득 확인

## 실패 시
- 재시도: 포인트 부족 → 상회 미션 수행으로 포인트 확보
- 복구: 구매 한도 초과 → 다음 갱신까지 대기

## 타이밍
- 대기: 800ms
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 400): 상점 탭
- tap(540, 700): 아이템 선택
- tap(700, 1000): 수량 조절
- tap(540, 1300): "교환" 버튼

## 관련 state
- guild_points: 보유 상회 포인트
- guild_level: 상회 LV (상점 품목 범위)
- daily_purchase_limit: 일일 구매 잔여
- weekly_purchase_limit: 주간 구매 잔여
