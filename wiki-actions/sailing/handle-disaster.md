# 재해 대응 (Handle Disaster)

## 목적
- 항해 중 발생하는 해상 재해를 감지하고 적절한 도구로 해소

## 전제조건
- 상태: Sailing
- 화면: 항해 화면 + 재해 알림 아이콘/팝업 표시

## 재해 종류 및 해소 도구
| 재해 | 도구 |
|------|------|
| 쥐 | 쥐약 |
| 오염, 화물 붕괴 | 청소도구 |
| 화재, 대화재 | 소화 모래 / 소화 세사 |
| 침수, 암초, 부패 | 통 |
| 괴혈병 | 라임 주스 |
| 욕구불만 | 술통 |
| 전염병 | 전염병 치료제 |
| 세이렌 | 귀마개 |
| 싸움, 반란, 화물 도난 | 징벌의 밧줄 |
| 불면증 | 해먹 |
| 향수병 | 망향의 카리욘벨 |
| 정서불안 | 뱃노래 악보 |
| 해초 | 긴 낫 |
| 폭풍 | 성스러운 향유 |
| 격랑, 눈보라, 돌풍 | 해신의 부적 |
| 무풍 | 풍신의 콩 |
| 자기장 이상 | 간이 나침반 |
| 식인상어 | 허수아비 |
| 실명 | 세안약 |
| 구속 | 톱 |

## 단계
1. 감지: 화면 상단/하단에 재해 아이콘(부정 효과) 표시 확인
2. 실행: [메뉴] → [도구] 또는 재해 아이콘 tap → 해당 도구 선택 → 사용
3. 확인: 재해 아이콘 소멸, 부정 효과 목록에서 제거

## 실패 시
- 도구 미보유: 가장 가까운 항구로 자동항해 → 도구점 구매
- 기적 사용 가능 시(초심자 혜택): 기적으로 전체 해소
- 방치 시 자연 해소 가능하나 피해 누적 주의

## 타이밍
- 대기: 500ms (메뉴 열기 후)
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(menu_btn): 메뉴 열기
- tap(tool_icon): 도구 카테고리 선택
- tap(cure_item): 해당 도구 사용

## 관련 state
- Sailing → (재해 감지) → Event → (해소) → Sailing
- 보급법 LV 높으면 일부 재해 자연 해소 확률 증가
