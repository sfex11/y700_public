# 스토리 퀘스트 수주 (Accept Story Quest)

## 목적
- 연대기 진행 중 발생하는 스토리 퀘스트를 수주한다

## 전제조건
- 상태: 연대기 챕터 진행 중
- 화면: 자동 발생 팝업 또는 NPC 대화 화면
- 해당 제독으로 플레이 중 (다른 제독으로는 수행 불가)

## 단계
1. 감지: 스토리 퀘스트 발생 팝업 또는 NPC 대화에서 퀘스트 아이콘 확인
2. 판단: 퀘스트 타입 확인 — 이동/전투/수집/교역/발견
3. 실행: "수주" 버튼 tap
4. 확인: 퀘스트 목록에 추가 확인, 목표 지점/조건 표시
5. 판단: 고유 항해사 합류 이벤트 트리거 여부 확인
   - 조안 → 로코 알렘켈, 피리 → 아흐메트 무히딘, 레온 → 카를로타 등

## 실패 시
- 재시도: 퀘스트 팝업 놓침 → 연대기 메뉴에서 재확인
- 복구: 수주 조건 미달 → 선단 LV/챕터 진행도 확인

## 타이밍
- 대기: 1000ms (퀘스트 수주 처리)
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 1000): 퀘스트 팝업 "수주" 버튼
- tap(540, 800): NPC 대화 "수락"

## 관련 state
- active_admiral: 현재 제독
- quest_type: 퀘스트 타입
- quest_target: 목표 (도시/적/아이템)
- chronicle_navigators: 합류 가능 고유 항해사
