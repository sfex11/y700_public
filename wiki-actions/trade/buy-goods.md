# 물품 매입

## 목적
- 교역소에서 시세가 낮은 물품을 매입한다 (목표: 시세 ≤90%)
- 흥정(협상 전략)으로 추가 할인 시도

## 전제조건
- 상태: 교역소 진입 완료, 잔여 적재량 확인 완료
- 화면: 교역소 매입 탭
- 골드 충분, 적재량 여유 있음
- 이슬람 문화권에서는 주류 매입 불가

## 단계
1. 감지: 매입 탭 선택 확인. 시세 ≤90% 품목 탐색
2. 실행: 대상 품목 tap → 수량 조절 → "매입" 버튼 tap. 흥정 가능 시 "구매 전략" 발동
3. 확인: 매입 완료 팝업 확인. 적재량 변화 검증. 관세 발생 여부 확인

## 실패 시
- 재시도: 적재량 부족 시 수량 줄여 재시도
- 복구: 골드 부족 시 매도 먼저 진행. 흥정 실패 시 그대로 매입 또는 스킵

## 타이밍
- 대기: 1000ms (매입 처리)
- 랜덤 딜레이: 300~600ms (흥정 애니메이션 포함)

## ADB 명령 예시
- tap(270, 200): 매입 탭 선택
- tap(540, 800): 물품 선택
- tap(700, 1200): 수량 증가 버튼
- tap(540, 1400): 매입 확인 버튼

## 관련 state
- cargo_remaining: 잔여 적재량
- gold: 보유 골드
- bargain_skill: 구매 전략 보유 여부
- tariff_exempt: 면세증 보유 여부
