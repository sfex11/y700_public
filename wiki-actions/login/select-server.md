# 서버 선택

## 목적
- 서버 목록에서 대상 서버를 선택하여 접속한다

## 전제조건
- 상태: 앱 로딩 완료
- 화면: 서버 선택 화면 표시됨

## 단계
1. 감지: 서버 목록 UI 출현 확인
2. 실행: 대상 서버 항목 tap → 접속 버튼 tap
3. 확인: 서버 접속 후 로그인 화면 또는 홈 화면 전환 확인

## 실패 시
- 재시도: 서버 선택 미반응 시 swipe로 목록 스크롤 후 재시도
- 복구: 서버 접속 실패 시 다른 서버 선택 또는 재시도

## 타이밍
- 대기: 3000ms (서버 접속 대기)
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(540, 700): 서버 항목 tap (위치는 서버 순서에 따라 상이)
- swipe(540, 1200, 540, 600): 서버 목록 스크롤
- tap(540, 1700): 접속 버튼

## 관련 state
- target_server: 설정된 서버명
- screen: server_select → login
