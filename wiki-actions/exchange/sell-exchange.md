# 거래소 판매 등록 (Sell Exchange)

## 목적
- 거래소에 아이템을 판매 등록하여 레드젬을 획득한다

## 전제조건
- 상태: 메뉴 접근 가능
- 화면: [메뉴] → [거래소] → 판매 탭
- 판매 가능 아이템 보유, 등록 수수료 (두카트) 보유

## 단계
1. 감지: 거래소 → "판매" 탭 선택 확인
2. 실행: 판매할 아이템 선택 → 수량 설정
3. 판단: 가격 설정 — 최소 10 레드젬 (선박은 티어별 상이). 시세 참고
4. 확인: 등록 수수료 확인 — 10 레드젬당 2,000 두카트
5. 실행: "등록" 버튼 tap → 수수료 지불 확인
6. 확인: 등록 완료 → 창고/선거에서 해당 아이템 제거 확인
7. 참고: 등록 기간 48시간. 판매 수수료 레드젬의 15%

## 미판매 처리
- 48시간 경과 시 반품 또는 재등록 선택

## 실패 시
- 재시도: 두카트 부족 (수수료) → 교역으로 확보
- 복구: 등록 취소 → 아이템 반환

## 타이밍
- 대기: 1500ms (서버 통신)
- 랜덤 딜레이: 300~500ms

## ADB 명령 예시
- tap(540, 400): 판매 탭
- tap(540, 700): 아이템 선택
- tap(540, 1000): 가격 입력 필드
- tap(540, 1300): "등록" 버튼

## 관련 state
- sellable_items: 판매 가능 아이템 목록
- ducats: 보유 두카트 (수수료용)
- listing_duration: 등록 잔여 시간
- sale_fee_rate: 판매 수수료율 (15%)
