# 선단 편성 (Manage Fleet)

## 목적
- 함대를 편성하고 선박/항해사를 배치한다

## 전제조건
- 상태: 항구 내 (docked) 또는 메뉴 접근 가능
- 화면: [메뉴] → [선단] → 편성 탭

## 단계
1. 감지: 선단 → "편성" 탭 선택 확인
2. 판단: 함대 선택 (1~4함대). 1함대=주함대(플레이어), 2~4함대=파견함대
3. 실행: 함대 내 선박 슬롯에 선박 배치 (선단 LV에 따라 1~5척)
4. 판단: 기함 지정 — 기함에 제독 배치
5. 실행: 각 선박에 항해사 배치 (assign-navigator 참조)
6. 판단: 함대 성향 설정 — 모험/교역/전투 (파견 보상에 영향)
7. 확인: 편성 완료 → 함대 스탯 요약 확인

## 파견함대 운용
- 2~4함대는 자동 운항 가능 — 성향에 따라 파견 보상 차이

## 실패 시
- 재시도: 선박 부족 → build-ship으로 건조
- 복구: 항해사 부족 → hire-navigator로 고용

## 타이밍
- 대기: 1000ms
- 랜덤 딜레이: 200~500ms

## ADB 명령 예시
- tap(200, 400): 1함대 탭
- tap(400, 400): 2함대 탭
- tap(540, 700): 선박 슬롯
- tap(540, 1300): "편성 완료" 버튼

## 관련 state
- fleet_count: 함대 수 (최대 4)
- fleet_level: 선단 LV (슬롯 수 결정)
- fleet_tendency: 함대 성향 (모험/교역/전투)
- flagship: 기함 지정 여부
