# 친구 시스템 (Friend Action)

## 목적
- 친구 추가/삭제, 응원, 함대 방문 등 소셜 활동 수행

## 전제조건
- 상태: 메뉴 접근 가능
- 화면: [메뉴] → [친구]

## 단계
1. 감지: 친구 메뉴 진입 확인
2. 실행 (추가): "친구 검색" → 닉네임/ID 입력 → "추가" tap
3. 실행 (응원): 친구 목록에서 대상 → "응원" 버튼 tap (하루 1회, 보상 획득)
4. 실행 (방문): 친구 선택 → "함대 방문" tap → 친구 선단 정보 확인
5. 실행 (삭제): 친구 선택 → "삭제" tap → 확인 팝업

## 실패 시
- 재시도: 응원 이미 완료 → 다음 일일 리셋 대기
- 복구: 친구 목록 가득 → 비활성 친구 삭제 후 추가

## 타이밍
- 대기: 800ms
- 랜덤 딜레이: 200~400ms

## ADB 명령 예시
- tap(540, 400): 친구 메뉴
- tap(700, 600): "응원" 버튼
- tap(540, 800): 친구 선택
- tap(540, 1300): "추가/삭제" 버튼

## 관련 state
- friend_list: 친구 목록
- cheer_available: 응원 가능 여부 (일일 리셋)
- friend_slots: 친구 최대 수
